Device Tree for Xiaomi Redmi A1 / Redmi A1+ / POCO C50
======================================================

Xiaomi Redmi A1, Redmi A1+ and POCO C50 (codenamed _"ice"_) are entry-level smartphones powered by the MediaTek Helio A22 platform.

The devices were released in 2022 and ship with Android 12 (Go Edition). Many firmware releases group Redmi A1, Redmi A1+ and POCO C50 together under the same codename, **ice**. 0
