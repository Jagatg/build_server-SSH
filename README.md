Device Configuration for Xiaomi Redmi A1 / Redmi A1+ / POCO C50
======================================================

Xiaomi Redmi A1, Redmi A1+ and POCO C50 (codenamed _"ice"_) are entry-level smartphones from Xiaomi.

Devices were announced and released in September 2022.

## Device specifications

Basic   | Spec Sheet
-------:|:-------------------------
SoC     | MediaTek Helio A22 (MT6761)
CPU     | Quad-core Cortex-A53 @ 2.0 GHz
GPU     | PowerVR GE8320
Memory  | 2/3 GB RAM
Shipped Android Version | Android 12 (Go Edition)
Storage | 32 GB eMMC 5.1
Battery | Non-removable Li-Po 5000 mAh battery
Display | 720 x 1600 pixels, 20:9 ratio, 6.52 inches, IPS LCD
